# Dr Bent Web

Página web profesional del Dr. Alvin Bent, especialista en ortopedia, hombro, rodilla y medicina deportiva.

## Tecnologías usadas:
- React
- Vite
- Tailwind CSS / PostCSS
- Firebase
- Stitch como origen de diseño
- Antigravity como entorno de desarrollo

## Comandos:
- `npm install` - Instalar dependencias
- `npm run dev` - Iniciar servidor de desarrollo local
- `npm run build` - Compilar proyecto para producción
- `npm run preview` - Vista previa del build local
- `npm run sync:github` - Sincronizar cambios locales con GitHub
